## ICT Order Block 核心理论

### 1. Order Block 的本质
- **定义**: 机构大单成交区域，价格后续重新测试时提供支撑/阻力
- **形成机制**: Smart Money在某价位区域大量买入/卖出，留下"订单痕迹"
- **重测试原理**: 价格回到Order Block时，未成交的订单继续发挥作用

### 2. Order Block 识别标准
**Bullish Order Block (看涨订单块)**:
- 特征：下跌中的最后一根阴线，随后价格强势上涨
- 逻辑：机构在此区域大量买入，阻止进一步下跌
- 重测试：价格回调至此区域时获得支撑

**Bearish Order Block (看跌订单块)**:
- 特征：上涨中的最后一根阳线，随后价格强势下跌  
- 逻辑：机构在此区域大量卖出，阻止进一步上涨
- 重测试：价格反弹至此区域时遇到阻力

### 3. Order Block 质量评估
**高质量Order Block特征**:
- **强势离开**: 形成后价格快速大幅移动 (Displacement)
- **清晰边界**: 有明确的高低点边界
- **量价配合**: 形成时有相应的成交量支持
- **时间新鲜**: 较新的Order Block通常更有效

### 4. Order Block 交易应用
**Entry Strategy (入场策略)**:
- **Optimal Trade Entry (OTE)**: 在Order Block的0.618-0.79斐波那契回撤区域入场
- **Fair Value Gap (FVG)**: 结合公允价值缺口寻找精确入场点
- **Market Structure**: 确认市场结构支持交易方向

## 必须分析的要素

### 1. Order Block 识别与标记

### 2. Market Structure 配合分析
- **BOS (Break of Structure)**: Order Block形成是否伴随结构突破
- **CHoCH (Change of Character)**: 市场特征改变对Order Block的影响
- **Internal Structure**: 小时间框架的结构支持

### 3. Order Block 重测试分析
- **First Time Back**: 首次回测通常最有效
- **Retest Quality**: 回测时的价格行为和成交量特征
- **Mitigation**: Order Block被完全消化的标志

### 4. 永续合约特殊性考虑
- **Funding Rate**: 资金费率对Order Block效力的影响
- **Leverage Effect**: 杠杆对Order Block形成和重测试的放大作用
- **24/7 Trading**: 连续交易对Order Block持续性的影响
